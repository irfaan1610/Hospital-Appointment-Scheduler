# 🏥 Slotify Health — Appointment Scheduling Web Application

Slotify Health is a fully responsive appointment scheduling web application that allows users to manage, visualize, and organize medical appointments efficiently through a clean calendar interface and dashboard.

---

## 🚀 Live Demo

👉 https://slotifyhealth.netlify.app/

---

## ✨ Features

### 📅 Calendar View

* Dynamic monthly calendar layout
* Accurate date alignment based on month/year
* Appointments displayed inside respective date cells
* Clear visual representation of scheduled entries

### 📝 Appointment Management

* Create appointments via modal form
* Edit existing appointments
* Delete appointments instantly
* Persistent data using browser localStorage

### 📊 Dashboard View

* Tabular display of all appointments
* Clean and structured layout for easy management

### 🔍 Search & Filter

* Search by patient name
* Search by doctor name
* Filter by date
* Real-time filtering without page reload

### 📱 Responsive Design

* Fully responsive across mobile, tablet, and desktop
* Optimized layout for different screen sizes
